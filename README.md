1.프로젝트 소개

프로젝트의 주제는 같은 그림 찾기 게임입니다.

다양한 난이도에서 제한 시간 안에 플레이어가 같은 그림을 찾는 게임입니다.

이 프로젝트는 Swift와 SwiftUI를 사용하여 iOS 앱으로 구현되었습니다.
